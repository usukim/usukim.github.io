## 기초 명령어

`dir`
: 현재 폴더의 파일과 하위 폴더 목록을 표시한다.

`cd [경로]`
: 현재의 작업 디렉토리 위치를 변경한다.

`cls`
: 명령 프롬프트 화면 내용을 모두 지운다.

`help`
: 사용 가능한 명령어 목록 및 도움말을 표시한다.

`exit`
: 명령 프롬프트를 종료한다.

## 파일 및 폴더 관리

`md [폴더명]`
: 새 폴더를 생성한다.

`rd [폴더명]`
: 폴더를 삭제한다. (폴더 내용물이 비어있어야 함)

`del [파일명]`
: 파일을 삭제한다.

`copy [원본] [대상]`
: 파일을 복사한다.

`move [원본] [대상]`
: 파일을 이동하거나 이름을 변경한다.

`ren [헌재 이름] [바꿀 이름]`
: 파일 이름을 변경한다.
